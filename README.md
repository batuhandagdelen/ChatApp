# 💬 ChatApp

Gerçek zamanlı mesajlaşma uygulaması. Firebase altyapısı üzerine inşa edilmiş, kullanıcıların anlık mesaj gönderebildiği, sohbet odaları oluşturabildiği ve profil yönetimi yapabildiği modern bir mobil uygulamadır.

## 🛠️ Kullanılan Teknolojiler

- **React Native** - Cross-platform mobil uygulama geliştirme
- **Expo (SDK 56)** - React Native geliştirme ortamı ve araçları
- **Expo Router** - Dosya tabanlı navigasyon sistemi
- **Firebase v12** - Authentication, Firestore veritabanı ve gerçek zamanlı veri senkronizasyonu
- **NativeWind v4** - Tailwind CSS tabanlı stil sistemi
- **React Native Reanimated** - Akıcı animasyonlar
- **React Native Async Storage** - Yerel veri saklama
- **@expo/vector-icons** - İkon kütüphanesi
- **TypeScript** - Tip güvenli geliştirme

---

## ⭐ Özellikler

- 🔐 **Kimlik Doğrulama** - E-posta & şifre ile kayıt ve giriş
- 💬 **Gerçek Zamanlı Mesajlaşma** - Firebase Firestore ile anlık mesaj senkronizasyonu
- 🏠 **Ana Ekran** - Aktif sohbet listesi ve kullanıcı yönetimi
- 🗨️ **Sohbet Odaları** - Kullanıcılar arasında birebir sohbet odaları
- 📋 **Mesaj Geçmişi** - Tüm mesajların kalıcı olarak saklanması
- 👤 **Profil Yönetimi** - Kullanıcı profil bilgilerinin düzenlenmesi
- 🔒 **Oturum Yönetimi** - Otomatik oturum açma/kapama (Context API)
- 📱 **Cross-Platform** - iOS, Android ve Web desteği
- 🎨 **Modern Tasarım** - NativeWind ile şık ve responsive arayüz
- ⚡ **Hızlı Performans** - Reanimated ile akıcı animasyonlar
